# How to run

0. Install gradle or `./gradlew wrapper --gradle-version=7.4 --distribution-type=bin`
1. build jar file with `./gradlew wrapper build`
2. run `docker-compose up -d`
3. access `http://localhost:8081/swagger-ui/index.html#/`
